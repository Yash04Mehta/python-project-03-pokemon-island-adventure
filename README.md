# Pokemon Island Adventure 🧭⚡

This is a beginner Python text-based adventure game where the player explores Pokemon Island and makes decisions to capture Mewtwo.

The game uses nested conditional logic and user input to create a branching storyline experience.

---

## Story Overview

You arrive at a mysterious crossroad on Pokemon Island.

Your mission:

Capture Mewtwo.

Along the journey you may encounter:

- Charizard 🔥
- Venusaur 🌿
- Blastoise 🌊
- Gyarados 🌊🐉
- Dugtrio 🪨

Only the correct path leads to success.

---

## How the Game Works

The player makes decisions at three key stages:

1. Choose direction at the crossroad
2. Choose whether to wait or swim at the lake
3. Choose a door inside the island house

Each decision changes the outcome of the story.

---

## Example Gameplay

Input:

left  
wait  
blue  

Output:

Mewtwo is yours  
Game Over.

---

## Concepts Used

This project demonstrates:

- nested if statements
- user input handling
- conditional branching
- multi-path decision logic
- ASCII art display in terminal
